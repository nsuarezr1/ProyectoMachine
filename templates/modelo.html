3. Ingeniería del Modelo

  • Algoritmo: Se utiliza el modelo de regresión lineal en este proyecto de CO2 en Colombia porque permite identificar y cuantificar las relaciones entre las emisiones y diversos factores en cada sector, facilitando la predicción de escenarios futuros y el análisis de tendencias históricas.
  • Entrenamiento: Para entrenar el modelo de Machine Learning capaz de pronosticar las emisiones de dióxido de carbono en Colombia hasta el año 2014, lo primero que se hace es reunir toda la información histórica de las emisiones de CO2 generadas por los diferentes sectores, utilizando el dataset del IPCC como base. Paralelamente, se recopilan los datos históricos de los posibles impulsores de estas emisiones en cada sector hasta 2014. Una vez que se cuenta con estos datos, el siguiente paso crucial es preparar y estructurar esta información de manera que pueda ser utilizada como base para el modelado predictivo.
