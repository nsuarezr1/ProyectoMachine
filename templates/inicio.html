1. Entendimiento del Negocio y los Datos

INTRODUCCIÓN: El objetivo primordial radica en desarrollar un modelo de Machine Learning capaz de pronosticar las emisiones de dióxido de carbono generadas por un sistema de las posibles futuras trayectorias de las emisiones de gases de efecto invernadero (GEI) utilizando este dataset del IPCC. Lo que radica en proporcionar perspectivas claras y accionables para comprender mejor los impulsores de las emisiones, identificar áreas clave para la mitigación y, potencialmente, construir herramientas predictivas que informen la toma de decisiones en materia de política climática y sostenibilidad. Se Busca convertir estos datos en conocimiento estratégico que pueda ser utilizado por diferentes organizaciones para abordar el desafío del cambio climático de manera más efectiva.

    OBJETIVOS
  • Crear una comprensión visual clara de cómo las emisiones de CO2, han cambiado a lo largo del tiempo para las diferentes categorías del IPCC.
  • Preparar y estructurar los datos de manera que puedan ser utilizados como base para futuros esfuerzos de modelado predictivo
    • PREGUNTAS CLAVE:
  • ¿Qué sectores generan más CO₂?, ¿Cómo evolucionan sus emisiones?
  • ¿Hay algún año o sector que muestre un comportamiento inusual en las emisiones que pueda requerir una investigación adicional sobre posibles causas?
    • EVALUACIÓN:
  • Es una estructura organizada por año y categoría del IPCC, con valores de emisiones para los tres gases principales. La consistencia en la presentación de las columnas a lo largo de los años es un punto positivo.
    • VIABILIDAD:
  • Se manifiesta una alta viabilidad para realizar un análisis exploratorio de datos (EDA) que permita comprender las tendencias históricas y las relaciones básicas dentro del dataset. Se puede crear visualizaciones para ilustrar la evolución de las emisiones y la contribución sectorial.La viabilidad para la predicción de emisiones futuras basándose únicamente en este dataset histórico es moderada.Para una predicción más robusta y útil para la toma de decisiones, sería necesario integrar datos socioeconómicos, de políticas energéticas, de desarrollo tecnológico y otros factores relevantes específicos para Colombia. Por lo tanto, el enfoque inicial se centrará en un análisis exhaustivo de los datos históricos para sentar una base sólida. La viabilidad de la predicción a largo plazo dependerá de la disponibilidad y la calidad de los datos predictivos adicionales que pueda obtener.
